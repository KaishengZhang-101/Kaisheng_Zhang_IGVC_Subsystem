# Kaisheng Zhang IGVC Subsystem Repository

This repository contains my individual GitHub submission for the IGVC Milk Dragon vehicle project.

## Subsystem Title

Camera Perception, Road Following, and Nav2-Based System Integration for an IGVC Autonomous Ground Vehicle

## Author

Kaisheng Zhang  
Milk Dragon Team  
University of Detroit Mercy

## Repository Organization

- `Folder1_My_Work/` contains my primary subsystem work, including user manuals, ROS 2 packages, launch files, configuration files, diagnostics, and system drawings.
- `Folder2_Collaborative_Work/` contains collaborative vehicle-level files that I used, tested, or integrated with my subsystem.
- `zip_design_files/` contains compressed design artifacts that are not source code, if applicable.

## Main Contributions

My main work included:

1. Camera-based lane perception integration.
2. Lane pixel to ground-coordinate projection.
3. Lane-center target generation.
4. Front-sector LiDAR filtering for navigation.
5. Nav2-based bring-up and local costmap integration.
6. Road-following behavior used in the final integrated system.
7. Diagnostics and startup documentation for repeatable robot operation.

## Notes on Authorship

Files in `Folder1_My_Work/` are my primary or substantially modified work unless otherwise marked.

Files in `Folder2_Collaborative_Work/` are collaborative system-level materials and are included only where I contributed to testing, integration, documentation, or subsystem connection.
